# LogSentinel.
**LogSentinel:** Tarayıcı tabanlı yerel siber güvenlik log analizörü. Ham sunucu günlüklerini (.log/.txt) üçüncü parti sunuculara göndermeden, JavaScript FileReader API ve Regex ile tamamen yerelde işler. Şüpheli IP adreslerini, Brute-Force sızma girişimlerini ve kritik sistem hatalarını yakalayıp dinamik bir siber panelde görselleştirir.
